# Notes App Frontend

## Tech stacks

- **Vue 3**
- **Vite**
- **TypeScript**
- **TailwindCSS**
- **Pinia**
- **Axios**
- **Deployment:** GitHub Pages

---

## Features

### Note Management

- Create a note (title + optional content)
- Read all notes
- View a single note
- Update an existing note
- Delete a note

## Environment variables

Create a `.env` file in the project root by copying:

```bash
cp .env.example .env
```

Then update:

```VITE_API_URL=<actual-backend-api-url>```

Example:

```VITE_API_URL=http://localhost:5000/api```

## Installation

### Node Version

_ ```nvm install```

_ ``` nvm use ```

### Install Package

_ ```npm install```

### Run the project

_ ```npm run dev```

### Deployment for github page

_```npm run build```

_ ```npm run deploy```
